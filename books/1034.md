# deepincpu不自动降频

解决方法：

1）安装i7z查看CPU频率的实时结果

```
sudo apt install i7z
sudo i7z
```

2）解决方法：

```
sudo gedit /etc/default/grub
```

3）编辑grub文件，其中两行改为如下：

```
GRUB_CMDLINE_LINUX="splash quiet "
GRUB_CMDLINE_LINUX_DEFAULT="intel_pstate=disable"
```

4）保存退出后更新一下grub

```
sudo update-grub
```
